# Popup

A popup to use on a website.

## Basic use

The popup is supposed to collect from users their decisions regarding personal data processing.
It fetches a text and a list of vendors. User will decide which vendor he accepts and his decision is saved to cookies.
The popup will appear after 24 hours.

## How to use

Clone the project or simply download it. Rightclick on the HTML file and open in your browser.
